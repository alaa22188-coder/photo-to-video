# 🎬 أداة تحويل الصور إلى فيديو مع صوت مجاناً (Photo to Video Converter)

### الإصدار المحدث والأنظف لعام 2026 - The Advanced Web Version for 2026

هذه هي الأداة الرسمية والاحترافية المصممة لتصميم ومزامنة الصور مع الملفات الصوتية بذكاء وسهولة عبر المتصفح مباشرة. تتيح لك الأداة ترتيب كليباتك، التحكم بوقت الظهور، وتصدر فيديوهاتك بجودة عالية للمقاسات العرضية والعمودية (Shorts) بدون تعقيد.

This is the official browser-based tool designed to seamlessly sync images with audio files. It allows you to arrange image clips, control duration precisely, and export high-quality videos in both Landscape and Vertical (Shorts) formats.

## 🚀 المميزات (Features)

* **صناعة وتعديل أونلاين:** يعمل بالكامل داخل المتصفح (Client-Side) بدون الحاجة لرفع ملفاتك إلى خوادم خارجية (أمان وخصوصية تامة).
* **دعم مقاسات متعددة:** يدعم الفيديو العرضي (16:9) والمقاس العمودي (9:16) المخصص للفيديوهات القصيرة (Shorts, Reels, TikTok).
* **مزامنة ذكية ووقت دقيق:** لوحة تحكم مرنة تتيح لك تحديد وقت بدء واختفاء كل صورة بالثواني، مع ميزة المزامنة الفورية مع اللحظة الحالية للمشغل.
* **تصدير فوري وبجودة عالية:** تصدير الفيديو مباشرة بصيغة `WebM` مع دمج الصوت بكفاءة عالية وبضغطة زر واحدة.

## 📖 طريقة الاستخدام (How to Use)

1. قم برفع ملف الصوت الخاص بك بصيغة (Audio).
2. أضف الصور التي تريد دمجها في الفيديو.
3. قم بجدولة ظهور واختفاء كل صورة عبر لوحة التحكم الزمني بالثواني.
4. اضغط على زر **"تصدير وفك فيديو فوري 🚀"** وانتظر اكتمال العداد.
5. استعرض الفيديو النهائي وقم بتحميله مباشرة إلى جهازك.
